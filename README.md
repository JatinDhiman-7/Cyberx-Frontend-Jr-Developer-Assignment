🩺 Doctor Listing App (Next.js)

A simple Doctor Listing web application built using Next.js (App Router) and React.
This project allows users to view, search, and filter doctors based on specialization.


✨ Features
📋 Doctor listing with name, specialization, and consultation fee
🔍 Search doctors by name
🎯 Filter doctors by specialization
📱 Fully responsive design (mobile + desktop)
⚡ Built-in API using Next.js Route Handlers
⏳ Loading state handling
❌ Basic error handling



🛠️ Tech Stack
Next.js (App Router)
React
Tailwind CSS
JavaScript 
Next.js API Routes
📡 API Implementation


This project uses a local API built inside Next.js using route handlers.
